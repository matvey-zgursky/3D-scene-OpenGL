# Построение волнистой поверхности
## Цель
Реализовать волнистую поверхность на основе математической функции, построить ее в виде сетки и отобразить в трехмерной сцене средствами OpenGL.
## Шаги
- [x] Выбрать математическую функцию для задания волнистой поверхности.
- [x] Реализовать функцию вычисления высоты поверхности по координатам.
- [x] Определить границы поверхности по осям `x` и `z`.
- [x] Выбрать шаг сетки для построения поверхности.
- [x] Сгенерировать вершины поверхности.
- [x] Организовать отрисовку поверхности в виде набора полигонов.
- [x] Проверить корректность формы поверхности.
- [x] При необходимости скорректировать диапазоны и шаг сетки.
