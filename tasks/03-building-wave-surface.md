# Построение волнистой поверхности
## Цель
Реализовать волнистую поверхность на основе математической функции, построить ее в виде сетки и отобразить в трехмерной сцене средствами OpenGL.
## Шаги
- [ ] Выбрать математическую функцию для задания волнистой поверхности.
- [ ] Реализовать функцию вычисления высоты поверхности по координатам.
- [ ] Определить границы поверхности по осям `x` и `z`.
- [ ] Выбрать шаг сетки для построения поверхности.
- [ ] Сгенерировать вершины поверхности.
- [ ] Организовать отрисовку поверхности в виде набора полигонов.
- [ ] Проверить корректность формы поверхности.
- [ ] При необходимости скорректировать диапазоны и шаг сетки.
