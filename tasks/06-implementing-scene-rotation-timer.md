# Реализация вращения сцены по таймеру
## Цель
Добавить автоматическое вращение сцены по таймеру, чтобы обеспечить динамический обзор поверхности и источника света, а также подготовить основу для последующего управления запуском и паузой вращения.
## Шаги
- [ ] Ввести переменную угла поворота сцены.
- [ ] Реализовать функцию таймера для периодического обновления состояния сцены.
- [ ] Увеличивать угол поворота через фиксированный интервал времени.
- [ ] Вызывать перерисовку сцены после обновления угла.
- [ ] Повторно запускать таймер внутри callback-функции.
- [ ] Применить поворот сцены в функции отрисовки.
- [ ] Проверить, что сцена вращается плавно и без ошибок.
