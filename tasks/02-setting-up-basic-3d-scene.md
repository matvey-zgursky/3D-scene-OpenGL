# Настройка базовой 3D-сцены
## Цель
Настроить базовую трехмерную сцену: задать цвет фона, включить буфер глубины, настроить проекцию и камеру, а также подготовить функцию отрисовки для дальнейшей работы с графическими объектами.
## Шаги
- [ ] Задать цвет фона сцены.
- [ ] Включить буфер глубины `GL_DEPTH_TEST`.
- [ ] Переключиться на матрицу проекции.
- [ ] Настроить перспективную проекцию.
- [ ] Переключиться на матрицу модели-вида.
- [ ] Настроить положение камеры.
- [ ] Реализовать базовую функцию очистки экрана и буферов.
- [ ] Подключить функцию отрисовки к display-callback.
- [ ] Проверить, что сцена корректно отображается в окне.
