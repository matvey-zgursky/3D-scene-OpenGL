# Настройка базовой 3D-сцены
## Цель
Настроить базовую трехмерную сцену: задать цвет фона, включить буфер глубины, настроить проекцию и камеру, а также подготовить функцию отрисовки для дальнейшей работы с графическими объектами.
## Шаги
- [x] Задать цвет фона сцены.
- [x] Включить буфер глубины `GL_DEPTH_TEST`.
- [x] Переключиться на матрицу проекции.
- [x] Настроить перспективную проекцию.
- [x] Переключиться на матрицу модели-вида.
- [x] Настроить положение камеры.
- [x] Реализовать базовую функцию очистки экрана и буферов.
- [x] Подключить функцию отрисовки к display-callback.
- [x] Проверить, что сцена корректно отображается в окне.
