# Добавление освещения сцены
## Цель
Подключить освещение фиксированного конвейера OpenGL, активировать точечный источник света и добиться видимой реакции поверхности на освещение.
## Шаги
- [ ] Включить систему освещения `GL_LIGHTING`.
- [ ] Активировать источник света `GL_LIGHT0`.
- [ ] Задать начальные ambient-параметры света.
- [ ] Задать diffuse-параметры света.
- [ ] Задать specular-параметры света.
- [ ] Установить позицию точечного источника света в пространстве.
- [ ] Передавать параметры света при отрисовке сцены.
- [ ] Проверить, что поверхность визуально реагирует на освещение.
