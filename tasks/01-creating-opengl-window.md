# Создание и запуск окна OpenGL
## Цель
Подготовить базовую основу проекта: подключить необходимые библиотеки PyOpenGL, инициализировать GLUT, создать окно приложения и убедиться, что программа корректно запускает OpenGL-окно без ошибок.
## Шаги
- [ ] Подключить библиотеки `OpenGL.GL`, `OpenGL.GLU`, `OpenGL.GLUT`.
- [ ] Инициализировать GLUT.
- [ ] Задать режим отображения окна.
- [ ] Установить размер окна.
- [ ] Задать позицию окна на экране.
- [ ] Создать окно OpenGL с заголовком.
- [ ] Настроить базовые callback-функции.
- [ ] Запустить главный цикл обработки событий.
- [ ] Проверить, что окно успешно открывается и не закрывается с ошибкой.
