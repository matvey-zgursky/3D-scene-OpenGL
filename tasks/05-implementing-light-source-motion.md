# Реализация движения источника света
## Цель
Реализовать анимацию движения точечного источника света по круговой траектории над поверхностью и визуализировать его положение в сцене.
## Шаги
- [ ] Определить радиус круговой траектории движения света.
- [ ] Задать высоту источника света над поверхностью.
- [ ] Ввести параметр угла или времени для движения света.
- [ ] Реализовать вычисление координат источника света по круговой траектории.
- [ ] Обновлять положение источника света при каждом кадре или по таймеру.
- [ ] Передавать новые координаты в параметры `GL_LIGHT0`.
- [ ] Добавить визуализацию источника света в виде небольшого шара.
- [ ] Проверить, что источник света плавно движется над поверхностью.
