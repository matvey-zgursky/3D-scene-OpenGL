# Описание проекта
Программа позволяет продемонстрировать имитацию движения источника света по произвольной поверхности. Она предоставлять пользователю возможность влиять на свойства поверхности и луча. Также полученная сцена должна вращаться по таймеру и использовать команды переопределения свойств материала.

# Полезные команды
- При запуске нового терминала активируй виртуальное окружение: venv\Scripts\activate  
- Для запуска программы для тестирования: python main.py

# Дополнительные инструкции
- Файл main.py содержит запуск программы
- Файл scene.py содержит реализацию 3D-сцены
- Файл window.py содержит создание окна и регистрацию GLUT-callback’ов
- Файл surface.py содержит реализацию поверхности
- Файл input_handler содержит логику обработки клавиатуры
- Файл light_source содержит логику источника света
